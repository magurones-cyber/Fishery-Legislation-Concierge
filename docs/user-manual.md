# 利用者マニュアル

## 基本操作

1. `/dashboard` で最近の更新資料、よく使うメニュー、質問導線を確認する。
2. `/ask` で質問する。音声入力を使う場合は「音声入力」を押し、ブラウザのマイク許可を確認する。
3. 個人情報らしき内容が検知された場合は、マスキング前後を確認してから送信する。
4. 回答では、結論だけでなく根拠カード、条文番号、ページ、引用箇所、注意表示を確認する。
5. 回答後は「役に立った」「根拠不足」「引用誤り」等を評価し、必要に応じて自由記述を残す。

## 案件・写真

案件詳細では、現地写真、施設写真、領収書、請求書、漁船、機器、排水状況、市場衛生状況を添付できます。写真に氏名、住所、車両番号、船名、位置情報が含まれる場合はアップロード前に確認してください。

## オフライン利用

PWAとして基本画面、最近閲覧した資料、お気に入り、保存済みチェックリスト、未送信メモのオフライン閲覧を想定します。機密資料のオフライン保存は組織設定に従います。
