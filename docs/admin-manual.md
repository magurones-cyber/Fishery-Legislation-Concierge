# 管理者マニュアル

## 資料更新

1. `/admin/documents` で更新期限切れ資料を確認する。
2. `/admin/documents/new` で新しい版を登録する。
3. OCRが必要な資料はOCRジョブを作成し、結果を確認、修正する。
4. OCR又は修正後テキストから再チャンク化し、Embeddingを再生成する。
5. `/admin/documents/[id]/versions` で旧版、新版、差分、影響範囲を確認する。

## FAQ改善

`/admin/analytics/faq-candidates` で質問頻度、類似質問、回答評価、根拠資料、未解決事項を確認します。FAQ候補は自動公開せず、管理者が根拠確認後に公開します。

## 利用状況分析

`/admin/analytics` で質問数、利用者数、未解決質問、回答評価、資料閲覧数、検索数、相談案件数、期限超過案件、更新期限切れ資料を確認します。

## テナント管理

`/admin/tenants` で自治体名、表示名、対象区域、カテゴリ、テンプレート、チェックリスト、利用者、管理者の分離方針を確認します。追加自治体は新しい `organization_id` で作成します。
