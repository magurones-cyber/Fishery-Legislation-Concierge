# 外部連携設計

## e-Gov法令API

法令ID、改正日、施行日、条文XML又はHTMLを取得し、既存資料との差分を作成します。利用にはAPI仕様確認とバッチ実装が必要です。

## 条例データベース・自治体公式Webサイト

自治体ごとに公開形式が異なるため、RSS、サイトマップ、HTML差分、PDF差分を確認します。スクレイピングは利用規約とrobots.txtを確認します。

## Google Drive / OneDrive

OAuth同意画面、Drive API又はMicrosoft Graph API、フォルダ権限、同期ログが必要です。機密資料の同期先と共有権限を定期監査します。

## 通知

メール通知はSMTP又はSendGrid等、カレンダー通知はGoogle/Microsoft APIを利用します。更新期限、主任者講習期限、保険期限、案件期限を通知対象にします。

## CSV

CSVインポートはカテゴリ、タグ、FAQ、利用者を対象にし、プレビュー、バリデーション、差分確認後に取り込みます。CSVエクスポートは監査ログ、案件、FAQ候補、利用統計を対象にし、権限確認を必須にします。
