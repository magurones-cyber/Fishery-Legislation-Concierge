## 運用横断: GitHub容量管理・外部保存

- GitHubにはコード、SQL、最小seed、設計文書、テストのみを保存する。
- 法令PDF、写真、音声、動画、DBダンプ、バックアップ、Embedding、実案件データはStorage又はPostgreSQLへ保存する。
- `lib/storage` のStorageAdapterをS3、Cloudflare R2、Backblaze B2等へ差し替え可能にする。
- `/admin/storage`、`/admin/backups`、`/admin/archives` を実DBへ接続し、容量、転送状況、復元状況を確認できるようにする。
- `npm run check:repo-size` をCIに組み込み、大容量ファイルや秘密情報の混入を防止する。
